1. What is the difference between a block and an inline tag? Give examples of each one.

  • Block elements take up the full width and push everything below them to a new line, like a paragraph.

  • Inline elements only take up the space they need and stay within the line, like a word in a sentence.

    2. What is normal flow in HTML?

    This is the default way elements are arranged on the page. Block elements stack on top of each other, while inline elements stay next to each other in a line.

    3. What does float do? Why do you need to clear the float?

    Float pushes an element to the left or right, allowing other elements like text to wrap around it. You need to "clear" the float to prevent it from affecting other parts of the layout.

    4. What do absolute, fixed, and relative do?

  • Relative: Moves an element from where it would normally be, slightly up, down, left, or right.

  • Absolute: Puts an element in a specific place, not following normal flow.

  • Fixed: Keeps an element in the same spot on the screen, even when scrolling.

    • How can you make nice navigation buttons for mobile devices?

    For mobile, create big, easy-to-tap buttons that fit the screen width, with space around them so they are not too close together. This makes it comfortable for users to navigate.
