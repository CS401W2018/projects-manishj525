What is the difference between an ID and a Class?

Since an ID is unique to one element on a webpage, it cannot be shared by two elements. You can style it in CSS by using the # symbol (#myId, for example) followed by the ID name.

Shared styles between elements on a webpage are possible when a class is applied to multiple elements.

What is the difference between a SPAN tag and a DIV tag?

An inline element called a SPAN is used to group short text passages together without generating a line break.

Larger content sections are connected together using a DIV, a block-level element that begins on a new line.

What is the difference between a DIV tag and HEADER, MAIN, FOOTER?

DIV: A general container that lacks a defined purpose.

HEADER: Provides navigation or introduction content.

MAIN: Identifies the page's primary material.

FOOTER: Includes copyright information and other footer elements.
