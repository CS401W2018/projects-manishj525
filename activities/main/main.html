1. Why do we need a viewport meta tag?

Viewport Tag: It ensures your website scales to fit mobile screens correctly, avoiding tiny text or zoomed-out layouts.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

CSS Priority: When the same style (like flex-direction) is set twice for an element, the last one in the CSS file takes effect.
