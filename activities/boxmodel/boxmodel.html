Block Tags

What is the difference between block tags and inline tags.

  • Block elements take up the full width and start on a new line.

  • Inline elements stay within the same line as other text or content.

    • What is the margin, border, and padding?

  • Margin: Space outside an element.

  • Border: The edge surrounding an element.

  • Padding: Space inside an element between its content and the border.

    • What are units of measurement you can use in CSS?

  • Absolute units: Fixed sizes like pixels, centimeters, or inches.

  • Relative units: Sizes that change based on the screen, like percentages or viewport sizes.

    • When would you use a figure tag?

  • Use the figure tag for grouping an image and its caption together.
