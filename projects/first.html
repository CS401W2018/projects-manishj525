Internet

a worldwide network of linked computers that exchange information using established guidelines.

Web

Web (World Wide Web): An online network of connected papers and media files that may be viewed with a web browser.

How are the Internet and Web different?

The Web is an Online-based network of connected stuff, or webpages, while the Internet is a global computer network. The Web is one of the services provided by the network of servers that is the Internet.

ASCII and Unicode

a standard for character encoding which encodes text and control characters with 7-bit byte values.

What is UTF-8

a Unicode variable-length encoding that supports a previous relationship with ASCII and uses one to four bytes per character.

URL

(Uniform Resource Locator): An URL or link that opens Web resources.

Domain Name

a visible address such as www.example.com that refers to an IP address.

HTTP

A system called HTTP (Hypertext Transfer Protocol) is used to send data and web pages over the Internet.

HTTPS

Hypertext Transfer Protocol Secure, or HTTPS, is an encoded type of HTTP that provides safe online communication.

IP Address

An IP address is a unique name that is specific to every device connected to a network.

DNS

The DNS (Domain Name System) is a tool used to find websites by turning domain names into IP addresses.

W3C Recommendations

Official web development rules and guidelines established by the World Wide Web Consortium (W3C).

XML, XHTML, HTML, HTML5

Extensible Markup Language, or XML, is a flexible, organized format for readable by machines data storage and transfer. Extensible Hypertext Markup Language, or XHTML, is a more limited form of HTML that follows XML language guidelines. The common markup language used for building and arranging webpages is called HTML (Hypertext Markup Language). HTML5, the most recent version of HTML, has additional features for graphics, multimedia, and mobile-first web design.
